## 🖥️ Running Locally

The portfolio is a single self-contained HTML file — no build or install needed:

```bash
# Clone the repository
git clone https://github.com/ricardolimaa29/portfolio.git
cd portfolio

# Open directly in your browser
open portfolio-ricardo.html          # macOS
xdg-open portfolio-ricardo.html     # Linux
start portfolio-ricardo.html        # Windows
```
